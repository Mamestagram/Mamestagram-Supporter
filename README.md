## Banchoリーダーボード
### 機能
BanchoのリーダーボードをMamestagramでも利用できる機能です。

以下の機能が利用可能です

* <ins>100人</ins>までのBanchoのリーダーボードを取得
* アバターの閲覧
* Mod別のランキング
* リプレイの再生
* リーダーボードへ掲載
